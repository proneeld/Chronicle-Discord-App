# Chronicle-Discord-App
Source code for Chronicle, a Discord bot for scheduling meetings in Voice Channels, getting information regarding Valorant Champions Tour Tier 1, and making friendly bets on their games.

# Features
* Be able to schedule meetings in a Discord Voice Channel, allowing you to tag participants and automatically give a 5-minute reminder to the mentioned participants
* Assign warnings to users who are not in the specified voice channel on the specified day and time, be able to view the warnings of the people in the server, and reset the warnings (admin only)
* Get information about the Valorant Champions Tour (VCT) Tier 1 scene, such as rankings by region and live/past/upcoming games
* Gamble on VCT Tier 1 games using fake money, allowing for a 2x payout if you win and nothing if you lose. Also gives out daily money if a user goes below a certain amount to make it more forgiving
